=================
lib_dzne_igblastn
=================

Overview
--------

Libary for using igblastn.

Installation
------------

To install lib_dzne_igblastn, you can use `pip`. Open your terminal and run:

.. code-block:: bash

    pip install lib_dzne_igblastn
